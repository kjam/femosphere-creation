## Implementation

This skill adds joke-making capabilities to the rage bot. The bot:
1. Knows the targets and their problematic statements
2. Crafts jokes in the rage bot's voice
3. Can generate fresh jokes or reference specific incidents
4. Keeps it sharp, funny, and unapologetic
