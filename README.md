# femosphere-creation


As part of the Feminist AI LAN Party at PyConDE/PyData Germany 2026, we brought a provacative idea and mission. What would it mean to create a femosphere? 

In this repository, you'll find artifacts of this question and you're very encouraged to continue creating, building and dreaming of a femosphere

- Agent Work: we experimented with OpenCode and Claude Code using local LLMs with coding skills (mainly Qwen 3.5-35B). In this folder are some of the artifacts we created (AGENT.md and several attempts to make new skills)
- Art: a collection of images created mainly via a "Prompting Diffusion models to make a Femosphere" workshop (one server was running ComfyUI)
- Planet: learning how to graph with Python, by building a Femosphere planet! (Contributed by Kristian Rother)
